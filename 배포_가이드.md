# 이투시오 클라우드 배포 가이드

Vercel + Render를 사용하여 클라우드에 배포하는 방법입니다.

## 배포 후 장점

- 팀원들이 URL만 접속하면 게임 가능
- Node.js 설치 불필요
- 프로젝트 다운로드 불필요
- 24/7 서버 가동

## 1단계: GitHub에 코드 올리기

### 1. Git 초기화 (이미 완료됨)
```bash
git init
```

### 2. 변경사항 커밋
```bash
git add .
git commit -m "Initial commit - Etoosio game with online multiplayer"
```

### 3. GitHub 저장소 생성
1. GitHub.com 접속
2. 새 저장소 생성 (이름: etoosio)
3. Public으로 설정
4. 저장소 URL 복사 (예: `https://github.com/username/etoosio.git`)

### 4. GitHub에 푸시
```bash
git remote add origin https://github.com/username/etoosio.git
git branch -M main
git push -u origin main
```

## 2단계: 프론트엔드 배포 (Vercel)

### Vercel 계정 생성
1. [vercel.com](https://vercel.com) 접속
2. GitHub 계정으로 로그인
3. "Add New" → "Project" 클릭

### 프로젝트 연결
1. GitHub에서 `etoosio` 저장소 선택
2. "Import" 클릭

### 배포 설정
- **Framework Preset**: Vite (자동 감지됨)
- **Build Command**: `npm run build`
- **Output Directory**: `dist`
- **Install Command**: `npm install`

### 배포
1. "Deploy" 버튼 클릭
2. 1-2분 후 배포 완료
3. 배포된 URL 복사 (예: `https://etoosio.vercel.app`)

## 3단계: 백엔드 배포 (Render)

### Render 계정 생성
1. [render.com](https://render.com) 접속
2. GitHub 계정으로 로그인
3. "New" → "Web Service" 클릭

### 프로젝트 연결
1. GitHub에서 `etoosio` 저장소 선택
2. "Connect" 클릭

### 배포 설정
- **Name**: `etoosio-server`
- **Region**: Singapore (또는 Seoul)
- **Branch**: `main`
- **Runtime**: Node
- **Build Command**: `npm install`
- **Start Command**: `node server.js`

### 환경 변수 추가
1. "Advanced" → "Add Environment Variable"
2. **Key**: `PORT`
3. **Value**: `3001`
4. "Save"

### 배포
1. "Create Web Service" 클릭
2. 2-3분 후 배포 완료
3. 배포된 URL 복사 (예: `https://etoosio-server.onrender.com`)

## 4단계: 환경 변수 설정

### Vercel 환경 변수 추가
1. Vercel 대시보드에서 프로젝트 선택
2. "Settings" → "Environment Variables"
3. "Add New"
4. **Key**: `VITE_SERVER_URL`
5. **Value**: Render 배포 URL (예: `https://etoosio-server.onrender.com`)
6. "Save"

### 재배포
1. Vercel에서 "Deployments" 탭
2. 최신 배포 옆 "..." → "Redeploy"
3. 환경 변수가 적용되어 재배포됨

## 5단계: 테스트

### 접속 테스트
1. Vercel 배포 URL 접속 (예: `https://etoosio.vercel.app`)
2. "온라인 플레이" 클릭
3. 방 만들기/참가하기 테스트

### 팀원 공유
팀원들에게 다음 URL 공유:
- **게임 접속**: `https://etoosio.vercel.app`
- 방 코드만 공유하면 됨

## 주의사항

### Render 무료 플랜 제한
- 15분 비활동 후 슬립 모드
- 첫 접속 시 30초 정도 지연
- 월 750시간 제한

### Vercel 무료 플랜
- 제한 없음
- 자동 SSL
- 글로벌 CDN

### 문제 해결
- **연결 안 됨**: Render 서버가 깨어났는지 확인
- **CORS 오류**: Render URL이 올바른지 확인
- **배포 실패**: 로그 확인 후 재배포

## 비용

- **Vercel**: 완전 무료
- **Render**: 무료 플랜 (개인용 충분)

## 완료 후

팀원들에게:
```
게임 URL: https://etoosio.vercel.app

방법:
1. 위 URL 접속
2. "온라인 플레이" 클릭
3. 방 코드로 참가
```

이제 팀원들이 Node.js 없이도 바로 게임을 할 수 있습니다!
