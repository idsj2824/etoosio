# 초보자를 위한 이투시오 시작 가이드

터미널 명령어를 몰라도 쉽게 시작할 수 있습니다!

## 1단계: 파일 찾기

프로젝트 폴더(`c:\Users\KYJ\Desktop\etoosio`)를 열면 다음 파일들이 보입니다:

- **`서버시작.bat`** - 서버 시작용
- **`게임시작.bat`** - 게임 실행용
- **`팀원_플레이_가이드.md`** - 상세 가이드

## 2단계: 방장이 먼저 시작

### 서버 시작
1. **`서버시작.bat`** 파일을 더블클릭
2. 검은색 창이 열리고 "Server running on port 3001" 메시지가 나올 때까지 기다림
3. **이 창을 게임 중 계속 켜두세요!** (닫으면 안 됨)

### 게임 시작
1. **`게임시작.bat`** 파일을 더블클릭
2. 브라우저가 자동으로 열리고 게임 화면이 나옴
3. "온라인 플레이" 버튼 클릭

## 3단계: 방 만들기

1. "방 만들기" 탭 선택
2. 인원 수 선택 (3명, 4명, 5명)
3. "방 만들기" 버튼 클릭
4. 나온 방 코드를 팀원에게 카톡/메시지로 보내기

## 4단계: 팀원들 참가

팀원들은 각자의 PC에서:

1. **`게임시작.bat`** 파일 더블클릭
2. "온라인 플레이" 버튼 클릭
3. "방 참가하기" 탭 선택
4. 방장에게 받은 방 코드 입력
5. "방 참가하기" 버튼 클릭

## 5단계: 게임 시작

모두 입장하면 방장이 "게임 시작" 버튼 클릭!

## 주의사항

- **서버 창을 절대 닫지 마세요** (게임 중 계속 켜두기)
- 인터넷 연결이 필요합니다
- 같은 WiFi 네트워크에서 하면 더 안정적입니다

## 문제가 생기면

**"파일을 찾을 수 없습니다" 오류:**
- 프로젝트 폴더가 올바른 위치에 있는지 확인
- 파일이 존재하는지 확인

**"서버가 시작되지 않습니다" 오류:**
- `서버시작.bat`을 다시 실행
- 검은색 창에 에러 메시지가 있는지 확인

**게임이 열리지 않음:**
- `게임시작.bat`을 다시 실행
- 브라우저가 차단되지 않았는지 확인
